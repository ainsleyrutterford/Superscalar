# Superscalar

## Instruction set

- `nop`
- `add  r1 r2 r3`
- `addi r1 r2 c`
- `sub  r1 r2 r3`
- `subi r1 r2 c`
- `mul  r1 r2 r3`
- `div  r1 r2 r3`
- `lw   r1 A(i)`
- `li   r1 c`
- `sw   A(i) r1`
- `beq  r1 r2 dst`
- `bne  r1 r2 dst`
- `bgt  r1 r2 dst`
- `bge  r1 r2 dst`
- `blt  r1 r2 dst`
- `ble  r1 r2 dst`
- `j    dst`
- `jr   r1`
- `jal  r1`
